## [0.1.0-alpha] - 2026-03-26

### Added
- **Core:** Initial release of the `ingestix` framework.
- **Macros:** `ingestix-derive` for procedural macro support.
- **Async-first:** Full support for asynchronous data ingestion using Tokio.
- **Worker Trait:** Standardized interface for implementing custom data processors.
- **Examples:** Included multiple examples (Basic, Advanced, and Network-based) to showcase framework usage.
